# CustomScienceContracts English Contract Pack

This optional pack replaces the German mission catalog with English mission
titles, descriptions, subcategories and checklist labels.

## Install

1. Install the matching main CustomScienceContracts release first.
2. Copy this pack's `GameData` folder into your Kerbal Space Program folder.
3. Allow it to replace:

```text
GameData/CustomScienceContracts/Contracts/
```

Only contract catalog files are replaced. The plugin, icons, settings and save
state are unchanged.

## Compatibility

This pack uses the same contract ids, prerequisites, rewards and checks as the
German catalog, so existing progress remains compatible.

It is built for SOL Quarter-Scale. Stock KSP configs are planned separately.
